# Installation

Debian:

```
# curl -s http://tracker.breedhub.net/bhid/install.sh | sh -s
```

Manually:

* Install Node 8 and system build tools
* Run **npm install -g bhid --unsafe-perm**

## Prepare and start

```
# bhidctl install
# systemctl enable bhid
# systemctl start bhid
```
